О проекте

Проект представляет собой информационную систему, предназначенную для оптимизации закупок товаров в розничной торговле. В основе лежит математическая модель, минимизирующая совокупные издержки предприятия: транспортные, складские и капитальные.

Система реализована на стеке: Python (Flask) для серверной логики, HTML/CSS/JS для клиентского интерфейса и SQLite для хранения данных. Расчёты выполняются с помощью библиотеки scipy.optimize.

Все основные функции доступны зарегистрированным пользователям, включая запуск модели, просмотр и анализ результатов.
